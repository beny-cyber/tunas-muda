# 🌱 Tunas Muda - Web Penjualan Bibit Tanaman

Website platform e-commerce yang dirancang untuk mempermudah masyarakat dalam mencari dan membeli bibit tanaman berkualitas (seperti tanaman buah, kayu, dan hias) secara online.

## 🚀 Fitur Utama
* **Katalog Produk:** Menampilkan daftar bibit berdasarkan kategori tanaman secara rapi.
* **SEO Optimized:** Struktur kode yang ramah terhadap mesin pencari agar mudah ditemukan di Google.
* **Responsive Design:** Tampilan web yang nyaman dan responsif saat diakses melalui HP maupun laptop.
* **Direct Order:** Terintegrasi langsung untuk pemesanan cepat via WhatsApp Admin.

## 🛠️ Teknologi yang Digunakan
* HTML5
* CSS3
* JavaScript
